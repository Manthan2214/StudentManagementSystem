# 🎓 Student Management System

## 🚀 Technologies Used
- ASP.NET MVC 
- MySQL
- Entity Framework Core
- Bootstrap (UI)

## ✨ Features
- Add, Edit, Delete Students (CRUD)
- Search & Filter
- Dashboard UI
- Export to Excel
